Política de Privacidad

Última actualización: Mayo 2026

1. Información que recopilamos

Recopilamos la siguiente información cuando utilizas nuestros servicios:

  • Nombre y correo electrónico (al registrarte o iniciar sesión)
  • Información de perfil de Google si usas el inicio de sesión con Google
  • Datos de uso y preferencias de tema (oscuro/claro)

2. Cómo usamos tu información

  • Para crear y gestionar tu cuenta
  • Para proveerte acceso a Sabina Optimizer y otros servicios
  • Para enviarte tu license key después de una compra
  • Para mejorar nuestros servicios y experiencia de usuario

3. Almacenamiento de datos

Tus datos se almacenan de forma segura en Supabase (plataforma de base de datos en la nube). No almacenamos contraseñas en texto plano. Las sesiones de usuario se gestionan mediante localStorage del navegador.

4. Servicios de terceros

Utilizamos los siguientes servicios de terceros:

  • Supabase — base de datos y autenticación
  • Google Identity Services — inicio de sesión con Google
  • Vercel — hosting del sitio web
  • Mercado Pago / PayPal — procesamiento de pagos (no manejamos datos de tarjetas)

5. Tus derechos

Puedes solicitar la eliminación de tus datos en cualquier momento escribiéndonos a hola@leleoficial.com.

6. Contacto

Si tenés preguntas sobre esta política, contactanos a hola@leleoficial.com.
