Las 26 optimizaciones

Cada plan desbloquea más poder. Todos los detalles con un clic.

Power Plan: Alto Rendimiento

Activa el plan de energía de alto rendimiento

Bajo

Establece el plan de energía en "Alto rendimiento" para máxima potencia de CPU y GPU, eliminando limitaciones de ahorro de energía.

powercfg -setactive 8c5e7fda-e8bf-4a96-9a85-a6e23a8c635c

Desactivar HPET

Reduce latencia y mejora FPS. Requiere reinicio.

Medio

Desactiva el HPET (High Precision Event Timer) del sistema para reducir la latencia de interrupciones y mejorar FPS en juegos.

bcdedit /deletevalue useplatformclock

Game Mode

Prioriza recursos para juegos

Bajo

Activa el Game Mode de Windows para priorizar recursos del sistema hacia tus juegos, mejorando FPS y estabilidad.

Set-ItemProperty ... AutoGameModeEnabled 1

Deshabilitar Xbox Game Bar

Elimina superposición de Xbox

Bajo

Desactiva la Game Bar de Xbox para liberar recursos del sistema y evitar superposiciones que afectan el rendimiento en juegos.

GPU Hardware Scheduling

Reduce latencia de GPU. Requiere reinicio.

Medio

Activa la programación de GPU acelerada por hardware para reducir latencia y mejorar el rendimiento gráfico general.

Timer Resolution

Ajusta temporizador del sistema para menor latencia.

Bajo

Ajusta la resolución del temporizador del sistema para reducir la latencia en juegos y aplicaciones en tiempo real.

Deshabilitar Nagle

Reduce ping en juegos online

Bajo

Deshabilita el algoritmo de Nagle para reducir la latencia de red en juegos online y aplicaciones en tiempo real.

DNS Cloudflare

Cambia DNS a 1.1.1.1 (más rápido y seguro)

Bajo

Configura los DNS de Cloudflare (1.1.1.1 y 1.0.0.1) para navegación más rápida, segura y privada.

Optimizar TCP/IP

Ajusta TCP para menor latencia de red

Bajo

Optimiza los parámetros TCP/IP del sistema para menor latencia de red y mejor rendimiento en conexiones online.

Limpieza del sistema

Limpia TEMP, Prefetch, Papelera, Update cache, DNS

Bajo

Limpia archivos temporales, Prefetch, Papelera de reciclaje, caché de actualizaciones y vacía DNS para liberar espacio y mejorar respuesta.

Telemetría y Rastreo

Apaga servicios de diagnóstico de Windows

Bajo

Desactiva servicios de telemetría y rastreo de Windows que consumen CPU, RAM y envían datos a Microsoft. Libera recursos del sistema.

Efectos Visuales Ultrarrápidos

Desactiva animaciones lentas del sistema

Medio

Desactiva animaciones visuales lentas de Windows como minimizar/maximizar ventanas y animaciones de la taskbar. Mantiene fuentes suaves y aspecto visual.

Optimizar Input

Raw Buffer, mouse acceleration OFF, prioridad foreground

Bajo

Optimiza la respuesta del mouse y teclado: aumenta buffers de datos, desactiva aceleración del mouse y prioriza ventanas en foreground para input más directo.

Remover Bloatware

Desinstala apps preinstaladas (Xbox, Cortana, Skype, Copilot...)

Medio

Elimina aplicaciones preinstaladas de Windows que consumen recursos innecesariamente, liberando espacio y mejorando el rendimiento general.

Desinstalar OneDrive

Elimina OneDrive completamente del sistema

Medio

Elimina OneDrive y su integración con el explorador de archivos, liberando recursos del sistema que consume constantemente en segundo plano.

Optimizar SSD/NVMe

Hibernación OFF, SuperFetch OFF, TRIM

Bajo

Optimiza unidades SSD/NVMe: desactiva hibernación para recuperar espacio, desactiva SuperFetch (innecesario en SSDs) y ejecuta TRIM para mantener el rendimiento.

Shader Cache GPU

Maximiza caché de shaders NVIDIA

Bajo

Maximiza el tamaño del caché de shaders de NVIDIA y ajusta la preferencia de GPU para máximo rendimiento en juegos.

Limpiar Drivers Fantasma

Escanea y detecta drivers huérfanos

Bajo

Detecta y lista drivers huérfanos o fantasmas en el sistema que pueden causar conflictos o consumir recursos innecesariamente.

Guía Monitor

Configurar Hz máximo y overdrive

Bajo

Guía para configurar la frecuencia máxima de tu monitor y activar overdrive desde el menú OSD para menor ghosting.

Bloquear Apps en Segundo Plano

Impide que apps innecesarias consuman recursos

Bajo

Bloquea la ejecución de aplicaciones en segundo plano que consumen CPU, RAM y datos innecesariamente, liberando recursos para tus juegos.

MSI Mode (GPU+NVMe+USB)

Activa MSI en dispositivos. Reduce latencia DPC.

Medio

Activa Message Signaled Interrupts (MSI) en GPU, NVMe y USB para reducir drásticamente la latencia DPC y mejorar el rendimiento en tiempo real.

Guía DPC Latency

Recomendaciones BIOS para mínima latencia

Bajo

Recomendaciones detalladas para configurar la BIOS: desactivar C-States, SpeedStep y Global C-States para reducir la latencia DPC al mínimo.

Guía RAM Timings

Recomendaciones personalizadas para BIOS

Bajo

Guía personalizada basada en tu hardware: calcula timings primarios (tCL, tRCD, tRP, tRAS) óptimos para tu RAM específica.

Guía Overclock + Undervolt

Guía personalizada CPU/GPU

Bajo

Guía personalizada basada en tu CPU: recomendaciones de overclock (ratio +1-2) y undervolt (-0.05V) para CPU, y Afterburner para GPU (Core +150, Mem +750).

Benchmark Rápido

CPU, RAM, Disco y Ping

Bajo

Ejecuta un benchmark rápido del sistema mostrando CPU, RAM disponible, espacio en disco y latencia de ping para medir el rendimiento antes y después.

Crear Punto de Restauración

Deshace todos los cambios si algo falla

Bajo

Crea un punto de restauración del sistema antes de aplicar optimizaciones para poder deshacer todos los cambios en caso de cualquier inconveniente.
